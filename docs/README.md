# Lumen Yoga Docs

This folder contains lightweight maintenance docs for the codebase.

## Files

- `architecture.md`: Current app structure and key data/interaction flows.
- `testing.md`: How tests are set up and what is currently covered.
- `maintenance-log.md`: Recent engineering changes and why they were made.

## Update policy

When behavior, architecture, or tooling changes, update the matching file in this folder in the same PR/commit.
